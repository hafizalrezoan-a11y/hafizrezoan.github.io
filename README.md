# hafizrezoan.online

Personal portfolio site for Hafiz Al Rezoan — GIS Analyst & Geospatial Researcher.

Hosted via GitHub Pages, served at https://hafizrezoan.online.

## Local preview

Open `index.html` directly in a browser, or run a tiny static server:

```
python -m http.server 8000
```

then visit http://localhost:8000.

## Deploy

Pushing to `main` automatically publishes via GitHub Pages.
